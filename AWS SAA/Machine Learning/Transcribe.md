Amazon Transcribe is a **speech-to-text service** that automatically converts audio into text using machine learning.

---

## 1. Core Concept

- Input: **Audio / speech**
- Output: **Text transcription**
- Powered by:
    - **ASR (Automatic Speech Recognition)**

---

## 2. Key Features

### 2.1 Speech to Text

- Converts spoken language into written text
- Works in **real-time (streaming)** or batch processing

---

### 2.2 PII Redaction

- Automatically detects and removes **Personally Identifiable Information (PII)**

Examples of PII:

- Names
- Phone numbers
- Social Security Numbers

Use case:

- Protect sensitive data in transcripts

---

### 2.3 Automatic Language Identification

- Detects **multiple languages** in the same audio
- Supports multilingual transcription

Example:

- English + French in the same recording

---

## 3. Use Cases

- Transcribe **customer service calls**
- Generate **subtitles / closed captions**
- Create **searchable media archives**
- Extract **metadata from audio/video content**

---

## 4. How It Works

1. Provide audio input (file or stream)
2. Transcribe processes using ASR
3. Output:
    - Text transcript
    - Optional:
        - Redacted content
        - Language detection

---

## 5. Important Notes

- Uses **ASR (Automatic Speech Recognition)**
- Supports:
    - **Real-time streaming**
    - **Batch transcription**
- Can:
    - **Remove PII automatically**
    - **Detect multiple languages**
