Amazon Comprehend is a **fully managed NLP (Natural Language Processing) service** that uses **machine learning** to analyze and understand text.

---

## Core Idea

- Comprehend = **understand text**
- If you see **NLP in the → think Comprehend**

---

## Key Features

### 1. Language Detection

- Identifies the **language** of the text

---

### 2. Entity Recognition

- Extracts:
    - People
    - Places
    - Organizations
    - Events
    - Brands

---

### 3. Key Phrase Extraction

- Identifies **important terms** in text

---

### 4. Sentiment Analysis

- Determines if text is:
    - Positive
    - Negative
    - Neutral
    - Mixed

---

### 5. Syntax Analysis

- Tokenization (breaking text into words)
- Parts of speech tagging

---

### 6. Topic Modeling

- Automatically **groups documents by topics**

---

## What It Does

- Takes **unstructured text**
- Converts it into **structured insights**

---

## Use Cases

- Customer feedback analysis
- Support ticket analysis
- Social media sentiment analysis
- Document classification
- Content organization

---

## Example

Input:

- Customer emails

Output:

- Sentiment (negative/positive)
- Key issues (keywords)
- Entities (product names, locations)
