## What is Amazon Textract?  
  
**Amazon Textract** is a **machine learning service** that automatically extracts text and structured data from documents.  
  
It goes beyond simple OCR (Optical Character Recognition)  
  
It can extract:  
- Text  
- Handwriting  
- Tables  
- Forms (key-value pairs)  
  
---  
  
## How It Works  
  
1. You upload a document (PDF, image, scan)  
2. Textract analyzes it using **AI / ML**  
3. It returns structured output (JSON-like data)  
  
---  
  
## Input Types  
  
Textract can process:  
  
- 📄 PDFs  
- 🖼️ Images (JPG, PNG)  
- 🧾 Scanned documents  
- 📝 Forms and tables  
  
---  
  
## Output Data  

Textract can extract:  
  
- Text content  
- Key-value pairs (forms)  
- Tables (rows and columns)  
- Metadata (structured response)  
  
Example outputs:  
- Date of birth  
- Document ID  
- Invoice number  
- Names, addresses  
  
---  
  
##  Key Capabilities  
  
### 1. Text Extraction  
- Extract raw text from documents  
  

### 2. Forms Extraction  
- Detects:  
  - Keys (e.g. "Name")  
  - Values (e.g. "Youssef")  
  
  
### 3. Table Extraction  
- Converts tables into structured data  
- Preserves rows and columns  
  
 
### 4. Handwriting Recognition  
- Can read handwritten text (depending on quality)
